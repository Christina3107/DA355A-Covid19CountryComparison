The following resources were used to create this website:

Charts.js

The Charts.js library is used to create a graphical representation of up to date Covid-19 data for two different countries.

Covid-19 API

The Covid-19 API is used to retrieve up to date statistics made available by Johns Hopkins University.

Select2

The Select2 library is used to create a searchable select element in order to facilitate the selection of countries.

Geonames

The Geonames-API is used to retrieve the name of the current location.

Country-json

The country-json data is used to populate the select list, retrieve population data and display a flag item for the current location.

Bootstrap etc.

Bootstrap, JQuery and Popper.js are used for styling, manipulating the DOM and adding a tooltip.
