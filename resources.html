The following resources were used to create this website:

Charts.js

The Charts.js library is used to create a graphical representation of Covid-19 datasets for two different countries.

Check it out!

Covid-19 API

The Covid-19 API is used to retrieve up to date statistics made available by Johns Hopkins University.

Check it out!

Select2

The Select2 library is used to create a searchable select element in order to facilitate the selection of countries.

Check it out!

Geonames

The Geonames-API is used to retrieve the name of the current location based on longitude and latitude.

Check it out!

Country-json

The country-json data repository is used to populate the select list, retrieve population data and display flag items.

Check it out!

Bootstrap etc.

Bootstrap, JQuery and Popper.js are used for styling, manipulating the DOM and adding tooltips.

Check it out!
